# 第十三周小组周计划

## 核心目标
本周小组工作的核心聚焦于**系统安全闭环**与**AI 智能化底座**的双重构建。一方面，通过引入用户认证与权限体系，解决系统长期裸奔的安全隐患；另一方面，完成 AI 模型接入与基础对话链路，为后续的智能诊断打下地基。同时，持续推进多集群环境下的稳定性验证。


## 关键里程碑
- **周三**：完成 AI 基础对话接口（`/api/v1/ai/chat`）与用户注册登录接口（`/api/v1/auth/*`）的初版开发。
- **周五**：完成前后端联调，验证“登录 -> 注册集群 -> AI 对话”的完整用户路径。
- **周六**：进行多集群环境下的压力测试，确保引入认证与 AI 后系统性能无明显下降。

## 风险评估
- **模型服务不确定性**：外部 LLM API 可能存在波动，需设计降级方案（如切回 Mock 模式）。
- **网络环境复杂性**：跨网段 SSH 连接可能因防火墙或超时失败，需准备备用连接方案（如反向代理）。
