**姓名:** 沈永佳

**本周核心目标:**
*   完成用户注册与登录功能的后端 API 开发。
*   与前端同学进行接口联调，确保认证流程跑通。

**详细任务计划:**

1.  **开发用户注册接口 (`/api/v1/auth/register`):**
    *   **任务描述:** 实现完整的用户注册逻辑。
    *   **具体步骤:**
        1.  创建 `User` 实体类，并使用 JPA 注解映射到数据库表。
        2.  创建 `UserRepository` 接口，继承 `JpaRepository`。
        3.  在 `UserService` 中创建 `register` 方法，包含以下逻辑：
            *   接收前端传递的用户名、密码等信息。
            *   校验用户名是否已存在。
            *   使用 BCrypt 对密码进行加密。
            *   将新用户信息存入数据库。
        4.  在 `AuthController` 中创建对应的端点，调用 `UserService`。
    *   **预计完成时间:** 周三

2.  **开发用户登录接口 (`/api/v1/auth/login`):**
    *   **任务描述:** 实现用户登录认证，并返回 Token。
    *   **具体步骤:**
        1.  引入 JWT 相关依赖（如 `jjwt`）。
        2.  创建一个 `JwtUtil` 工具类，用于生成和解析 Token。
        3.  在 `UserService` 中创建 `login` 方法，包含以下逻辑：
            *   根据用户名查询用户。
            *   校验密码是否正确。
            *   如果认证成功，使用 `JwtUtil` 生成一个包含用户信息的 Token。
        4.  在 `AuthController` 中创建对应的端点，调用 `UserService`，并将生成的 Token 返回给前端。
    *   **预计完成时间:** 周四

3.  **接口联调:**
    *   **任务描述:** 与前端开发同学配合，进行注册和登录接口的联调测试。
    *   **具体步骤:**
        1.  使用 Postman 或类似工具，确保两个接口在后端独立运行时功能正常。
        2.  与前端同学约定时间，进行实时联调，解决可能出现的跨域（CORS）、数据格式不匹配等问题。
    *   **预计完成时间:** 周五

**学习目标:**
*   深入理解 RESTful API 设计规范。
*   熟练掌握 JWT 在 Spring Boot 项目中的应用。
