# 前端页面功能列表

本文档详细介绍了 `frontend-vue` 项目中包含的所有页面及其对应功能，供开发内容列表填写使用。

## 1. 认证与基础页面

- **登录页 (Login)**
  - **路径**: `/login`
  - **功能**: 用户身份验证，支持账号密码登录，包含错误处理及自动跳转逻辑。
- **注册页 (Register)**
  - **路径**: `/register`
  - **功能**: 新用户账号注册，包含表单校验和后端实时查重。

## 2. 集群与资源管理

- **集群列表 (Cluster List)**
  - **路径**: `/cluster-list`
  - **功能**: 展示所有纳管集群的状态、配置信息，支持集群的注册、编辑和删除操作。
- **系统总览 (Dashboard)**
  - **路径**: `/dashboard`
  - **功能**: 通过 ECharts 图表展示系统关键指标（如 CPU、内存使用率）、告警统计及集群健康度。

## 3. 日志与监控

- **系统日志 (Logs)**
  - **路径**: `/logs`
  - **功能**: 汇总展示集群及节点的运行日志，支持按级别和关键词搜索。
- **执行日志 (Exec Logs)**
  - **路径**: `/exec-logs`
  - **功能**: 专门展示自动化脚本或运维指令的执行历史及详细输出结果。
- **审计日志 (Audit Logs)**
  - **路径**: `/audit-logs` (仅管理员)
  - **功能**: 记录用户在系统内的所有操作行为，用于安全溯源。

## 4. 诊断与自动化

- **故障诊断 (Diagnosis)**
  - **路径**: `/diagnosis` (管理员/运维)
  - **功能**: 核心运维页面。支持实时日志流、节点故障排查树、以及集成 AI 助手进行智能化问题分析。
- **告警配置 (Alert Config)**
  - **路径**: `/alert-config` (管理员/运维)
  - **功能**: 配置系统监控指标的阈值及通知渠道。

## 5. 用户与账户管理

- **用户管理 (User Management)**
  - **路径**: `/user-management` (仅管理员)
  - **功能**: 管理系统用户信息，包括权限分配、账号状态切换等。
- **个人中心 (Profile)**
  - **路径**: `/profile`
  - **功能**: 展示当前登录用户的详细信息。
- **账号设置 (Account)**
  - **路径**: `/account`
  - **功能**: 修改密码、更新个人联系方式等账号安全操作。
