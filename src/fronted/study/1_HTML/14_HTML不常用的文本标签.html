HTML不常用的文本标签学习

1. <cite> 标签 - 引用作品标题

用途：用于标记对某个作品（如书籍、文章、电影、歌曲等）的引用。

示例：

我最近在读《JavaScript高级程序设计》这本书。

电影《肖申克的救赎》是我最喜欢的电影之一。

这首《月亮代表我的心》是经典老歌。

2. <dfn> 标签 - 定义术语

用途：用于标记术语的定义实例，通常是该术语在文档中首次出现时使用。

示例：

HTML（HyperText Markup Language）是用于创建网页的标准标记语言。

CSS是层叠样式表的缩写，用于描述HTML文档的样式。

响应式设计是一种网页设计方法，使网页能够在不同设备上良好显示。

3. <kbd> 标签 - 键盘输入

用途：用于表示用户应该从键盘输入的文本，如快捷键、命令等。

示例：

按下 Ctrl + C 复制选中的文本。

使用 Ctrl + V 粘贴内容。

在终端中输入 npm install 来安装依赖包。

按 F12 打开浏览器开发者工具。

4. <samp> 标签 - 程序输出示例

用途：用于标记计算机程序的输出示例或样本文本。

示例：

当你运行程序时，可能会看到以下输出：

Hello, World!

错误信息通常如下所示：

Error: Cannot find module 'express'

成功安装后会显示：

Package installed successfully!

5. <var> 标签 - 变量

用途：用于标记数学表达式或编程上下文中的变量。

示例：

在数学中，如果 x = 5，那么 y = x + 3 = 8。

函数 f(x) = x² + 2x + 1

在编程中，变量 userName 存储用户的姓名。

循环变量 i 从 0 开始递增。

6. 其他不常用文本标签

6.1 <abbr> 标签 - 缩写

用途：用于标记缩写或首字母缩略词。

示例：

WWW是万维网的缩写。

API是应用程序编程接口。

CSS用于网页样式设计。

6.2 <time> 标签 - 时间

用途：用于标记时间或日期，便于搜索引擎和其他程序理解。

示例：

这篇文章发布于 2024年1月15日。

会议时间：2024年1月20日下午2点。

项目截止日期：2024年2月1日。

6.3 <mark> 标签 - 高亮标记

用途：用于高亮显示文本，表示相关性或重要性。

示例：

在搜索结果中，关键词会被高亮显示。

请注意：这是重要信息，需要特别关注。

文档中的更新内容用高亮标记。

6.4 <small> 标签 - 小号文本

用途：用于表示附属细则，如版权信息、法律条文等。

示例：

欢迎使用我们的服务！使用条款和隐私政策适用。

产品价格：￥299 （不含税）

© 2024 我的网站. 保留所有权利。

6.5 <sub> 和 <sup> 标签 - 下标和上标

用途：用于表示下标和上标文本，常用于数学公式和化学方程式。

示例：

水的化学分子式是 H2O。

爱因斯坦的质能方程：E = mc2。

数学表达式：x2 + y2 = z2。

二氧化碳的分子式：CO2。

总结：这些不常用的HTML文本标签虽然使用频率不高，但在特定场景下能够提供更准确的语义化标记，有助于提升网页的可访问性和SEO效果。
